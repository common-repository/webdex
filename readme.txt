== Description ==

WebDex iti ofera acces instantaneu la 27 de dictionare ale limbii romane. Include widgetul WebDex in blogul sau site-ul tau si vizitatorii tai vor avea acces la aceste dictionare direct din pagina ta.

== Installation ==

1. Descarca fisierul webdex.php in directorul `/wp-content/plugins/'
2. Activeaza plugin-ul din meniul 'Plugins' al interfetei de administrare WordPress
3. Din sectiunea 'Appearance' > 'Widgets', selecteaza widget-ul WebDex si trage-l cu mouse-ul in locatia dorita
